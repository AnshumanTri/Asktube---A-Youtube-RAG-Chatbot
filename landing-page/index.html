The Process

How it works

Four steps from YouTube URL to grounded, cited answers — in seconds.

01
📺

Open any video

Navigate to any YouTube video with captions. The extension auto-detects the video — no copy-pasting URLs.

02
⚡

Index in one click

Hit "Index this video." The transcript is fetched, chunked with sentence-aware splitting, embedded locally, and stored in a FAISS vector index.

03
🤖

Agent retrieves

Your question is rewritten for better retrieval, then a LangGraph agent runs hybrid BM25 + MMR semantic search with contextual compression.

04
💬

Grounded answer

Llama 3.3 70B generates a detailed, tutor-style answer strictly from the transcript — with clickable timestamp citations to jump back in the video.

Under the Hood

Built different.

Not a wrapper around a single LLM call. A real agentic RAG system with a proper retrieval pipeline.

Agent

LangGraph Agentic Loop

A real state-machine agent that rewrites queries, retrieves, evaluates context quality, and retries — up to 3 iterations before answering. Not a single-shot chain.

LangGraph LangChain StateGraph
Retrieval

Hybrid BM25 + MMR Search

Sparse keyword search (BM25) fused with dense semantic MMR via Reciprocal Rank Fusion, then compressed by an LLM to extract only the relevant passage per chunk.

FAISS BM25 MMR RRF
Generation

Guardrails + Citations

Prompt injection detection, strict grounding rules, and structured timestamp citations baked into every response. The LLM cannot answer from outside the transcript.

Groq Llama 3.3 70B FastAPI
Evaluation

RAGAS Evaluated

Context precision, context recall, faithfulness, and answer relevancy measured on a held-out test set using RAGAS with a separate lightweight judge model.

RAGAS Llama 3.1 8B HuggingFace

Get Started

Install in 3 steps.

The backend is already live. You just need the extension.

1

Download the extension

Clone or download the repo from GitHub. The extension files are in the extension/ folder.

2

Load into Chrome

Go to chrome://extensions, enable Developer mode, click Load unpacked, and select the extension/ folder.

3

Open any YouTube video and start chatting

Click the TubeRAG icon in your Chrome toolbar while on a YouTube video. Hit Index this video, then ask anything.
